# WXT + React

1. Create a .env file with the trial token from chrome. 

```
\\ .env

summ_token = abcdefghijkelmn
```

2. Install packages and run the extension. 

1. ``npm install``

2. ``npm run dev``

3. A chrome window with the extension installed should open up.
